### Tab-Go

Inspired by [Capslock+](https://capslox.com/capslock-plus/)

### Feature

#### HotKeys

|       Tab +        | Mapping                                                      |
| :----------------: | ------------------------------------------------------------ |
| `i`, `j`, `k`, `l` | Arrow Keys: `up`, `left`, `down`, `right`                    |
|      `p`, `;`      | `Home`, `End`                                                |
|        `r`         | Tab-Go Function: reload the application (for developing purpose) |
|        `q`         | Tab-Go Function: exit the application                       |
